# 时光管理 - 任务与专注 (PomoReward)
一款结合番茄钟、任务管理和积分奖励机制的时光管理工具，帮助你高效专注、完成任务并实现心愿。

版本：v1.0.0
技术栈：HTML | TailwindCSS | JavaScript
授权协议：MIT

## 核心功能
### 专注番茄钟
- 自定义专注时长（1-120分钟）和休息时长（1-30分钟）
- 多专注模块支持（默认、阅读、编程等），可自定义模块名称和每分钟积分奖励
- 实时进度环展示，专注完成自动获得对应积分
- 专注结束自动提示，清晰的状态切换（专注中/休息中）

### 任务管理
- 支持添加普通任务和习惯任务
- 习惯任务可设置：
  - 每日重复（永久）
  - 限时完成（自定义持续天数）
- 任务完成自动获得积分奖励
- 习惯任务每日自动重置状态

### 心愿清单
- 添加心愿并设置所需积分
- 用完成任务和专注获得的积分兑换心愿
- 心愿兑换记录追踪

### 模块管理
- 自定义专注模块，设置不同模块的积分奖励
- 编辑/删除现有模块，适配不同专注场景

### 积分系统
- 专注时长积分（按分钟计算）
- 任务完成积分奖励
- 心愿兑换积分扣除
- 积分数据本地持久化存储

## 界面特色
- 清新的纸质纹理设计风格
- 响应式布局，适配PC和移动端
- 玻璃拟态效果，提升视觉体验
- 流畅的过渡动画和交互反馈
- 中文友好的界面设计

## 快速开始
### 在线使用
直接打开 "专注台V1.0.5.每分钟加积分.html" 文件即可使用，无需安装任何依赖。

### 本地部署
1. 克隆仓库：
git clone https://github.com/freytesla/PomoReward.git
cd PomoReward

2. 直接打开HTML文件：
   - 双击 "专注台V1.0.5.每分钟加积分.html"
   - 或使用浏览器打开该文件

3. 开始使用：
   - 无需配置，直接使用所有功能
   - 数据存储在浏览器本地存储中

## 适配说明
- 移动端：上下布局（番茄钟在上，任务/心愿在下）
- PC端：左右布局（番茄钟在左，任务/心愿在右）
- 适配主流浏览器（Chrome、Firefox、Safari、Edge）

## 数据存储
所有数据（任务、心愿、积分、模块设置等）均存储在浏览器的 localStorage 中：
- 数据持久化，关闭页面后不会丢失
- 支持全局重置功能，一键恢复初始状态
- 数据格式：JSON

## 技术栈
- HTML5（语义化标签）
- Tailwind CSS（样式框架）
- Font Awesome（图标库）
- JavaScript（原生，无框架）
- 本地存储（localStorage）

## 使用指南
### 基础操作
1. 设置专注时长：在番茄钟设置区域调整专注/休息时长
2. 选择专注模块：从模块列表中选择，或自定义新模块
3. 开始专注：点击"开始"按钮，专注结束自动获得积分
4. 添加任务：填写任务名称、积分奖励、任务类型
5. 完成任务：勾选任务复选框，自动获得积分
6. 添加心愿：填写心愿名称、所需积分、描述
7. 兑换心愿：积分足够时点击"兑换心愿"按钮

### 高级功能
- 管理专注模块：点击"管理模块"按钮，自定义模块设置
- 全局重置：点击右上角"重置"按钮，恢复初始状态
- 查看说明：点击左上角"说明"按钮，查看详细使用说明
