# 专注时光 - 待办与番茄钟 (PomoReward)
一款结合番茄钟、任务管理和积分奖励机制的专注工具，帮助你高效专注、完成任务并实现心愿。

版本：v1.1.0
技术栈：HTML | TailwindCSS | JavaScript
授权协议：MIT

## 核心功能
### 专注番茄钟
- 自定义专注时长（1-120分钟）和休息时长（1-30分钟）
- **🆕 自由模式**：支持正计时模式，不受固定时长限制，进度条始终100%
- **🆕 双模式切换**：可随时在番茄钟倒计时模式和自由正计时模式之间切换
- 多专注模块支持（默认、阅读、编程等），可自定义模块名称和每分钟积分奖励
- 实时进度环展示，专注完成自动获得对应积分
- **🆕 智能状态切换**：一键在专注和休息状态间切换，操作更便捷
- 专注结束自动提示，清晰的状态切换（待开始/专注中/休息中）

### 任务管理
- 支持添加普通任务和习惯任务
- 习惯任务可设置：
  - 每日重复（永久）
  - 限时完成（自定义持续天数）
- 任务完成自动获得积分奖励
- 习惯任务每日自动重置状态

### 心愿清单
- 添加心愿并设置所需积分
- 用完成任务和专注获得的积分兑换心愿
- 心愿兑换记录追踪

### 模块管理
- 自定义专注模块，设置不同模块的积分奖励
- 编辑/删除现有模块，适配不同专注场景

### 积分系统
- 专注时长积分（按分钟计算，番茄钟和自由模式均支持）
- 任务完成积分奖励
- 心愿兑换积分扣除
- 积分数据本地持久化存储

## 使用指南
### 基础操作
1. **设置专注时长**：在番茄钟设置区域调整专注/休息时长
2. **选择计时模式**：
   - 默认番茄钟模式：倒计时，时间到自动切换
   - 自由模式：勾选"自由模式"复选框，正计时，无时间限制
3. **选择专注模块**：从模块列表中选择，或自定义新模块
4. **开始专注**：点击"开始专注"按钮，进入专注状态
5. **切换状态**：再次点击按钮在专注和休息状态间切换
6. **添加任务**：填写任务名称、积分奖励、任务类型
7. **完成任务**：勾选任务复选框，自动获得积分
8. **添加心愿**：填写心愿名称、所需积分、描述
9. **兑换心愿**：积分足够时点击"兑换心愿"按钮

### 高级功能
- **管理专注模块**：点击"管理模块"按钮，自定义模块设置
- **全局重置**：点击右上角"重置"按钮，恢复初始状态
- **查看说明**：点击左上角"说明"按钮，查看详细使用说明
- **灵活切换模式**：根据任务需求随时在番茄钟和自由模式间切换

### 模式选择建议
#### 番茄钟模式适用场景：
- 需要严格时间管理
- 养成固定工作节奏
- 短时间高强度任务
- 需要定时休息提醒

#### 自由模式适用场景：
- 深度工作需要长时间专注
- 任务时间不确定
- 想要灵活控制休息时长
- 创意工作或心流状态

## 常见问题
**Q: 自由模式和番茄钟模式有什么区别？**
A: 番茄钟模式是倒计时，时间到自动切换；自由模式是正计时，进度条始终100%，适合不想被时间限制的场景。

**Q: 如何在两种模式间切换？**
A: 勾选或取消"自由模式"复选框即可，专注和休息可分别设置。

**Q: 自由模式下还能获得积分吗？**
A: 可以！自由模式下专注状态依然按分钟累积积分。

**Q: 按钮文字为什么会变化？**
A: 按钮文字反映下一步操作，如当前在专注中，按钮显示"开始休息"；当前在休息中，按钮显示"开始专注"。
